# CaseFlow video package

- `CaseFlow_Product_Demo.mp4` — 2:09 submission walkthrough (1920×1080). Real product states captured
  from the running app (replay of genuine recorded agent runs), narrated to match the real agent runtime.
- `CaseFlow_Product_Demo.srt` — caption track.
- `CaseFlow_Product_Demo_Narration.txt` — exact voiceover copy for re-recording in the pitch owner's voice.

All frames show synthetic CaseFlow data. Automated outputs are described only as recommendations; policy
activation remains human-controlled; evaluation language is limited to the measured synthetic-harness
results in `EVALUATION_REPORT.md` (90% disposition agreement, 100% citation recall — not production claims).

**Live alternative:** the app itself streams a live agent run (or a badged recorded run in Replay mode),
which is a moving screen recording available on demand during the pitch.

The included voice track is a temporary macOS system voice; replace it with the pitch owner's voice before
the pitch if time permits. Narration is timed to the existing edits.
